# Fast React Pizza Co.

A simple React app showcasing a pizza menu with conditional rendering and dynamic UI elements.

## Features

- Displays a pizza menu with prices and ingredients
- Handles open/closed store hours
- Modular React components
- Basic CSS styling

## Setup


     npm install
   
     npm start


## Tech Stack
 
     React

     JavaScript

     CSS
